Der Grund für diese Website

Diese Website entstand aufgrund einer Challenge, welche ich mir selber gestellt habe. Mit den Grundlagen der Webentwicklung habe ich mich ab Sommer 2019 beschäftigt, meine erste Homepage habe ich am 25. August 2019 auf GitHub veröffentlicht. Seitdem beschäftige ich mich durchgehend mit der Webentwicklung und dem Webdesign, weshalb ich mich auch dazu entschieden habe, die Website so umfangreich zu gestalten, wie es für mich möglich ist. Das Wissen & die Anwendung habe ich mir weitesgehend selber beigebracht, weshalb ich stetig Änderungen und Verbesserungen vornehmen werde.



FAQ

Häufig gestellte Fragen:
Wurde die Website mit einem Homepage-Baukasten erstellt?
Nein, die Website wurde von Grund auf selbst programmiert.
Seit wann gibt es diese Website schon?
Am 25 August 2019 habe ich das erste Mal etwas unter dieser Domain veröffentlicht.
Über welchen Anbieter hast du eine Website gehostet?
Netlify greift auf den Code in GitHub zu und hostet für mich die Website [Stand 23.03.2020].
Kann ich auch so eine Website programmieren?
Ja, natürlich. Wenn du konstant und motiviert am Projekt arbeitest, kann nur etwas gutes dabei rauskommen :)
